## Modules Covered

- **Registration** — positive and negative scenarios (login/password validation)
- **Login** — valid and invalid credentials
- **Logout** — session termination
- **Authorization Check** — access control for protected pages
- **Product List** — pagination, product card display
- **Add to Cart** — quantity changes, cart updates

**Status values:** Passed / Failed  
**Linked bugs:** Bug IDs referenced where applicable
