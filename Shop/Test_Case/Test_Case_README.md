# 🧪 Test Cases — demoshopping.ru

Functional test cases for an online electronics store [demoshopping.ru](http://demoshopping.ru).  
Created and managed in **Qase TMS**, exported as HTML files.

**Type:** Manual, Functional, E2E  
**Total test cases:** 10

Each test case includes:
